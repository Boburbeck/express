
# Express test exercise



## Prerequisites
This dummy project has following prerequisites
- Python 3.8.5
- Pytest 6.2.2


## Setup

- Install virtual environment:
```
git clone https://github.com/Boburbeck/express.git
cd express
python3.8 -m venv --prompt="v" .env
```

- Please install by running following command:
```
pip install -r requirements.txt
```

- Run tests:
```
pytest tests.py
```
